Tibia Item Market + Loot Finder

Local Python version: weekly items, Tibia prices for Bona/Celesta/Dia, Grizzly Adams tasks, and imbuing material prices.

Search

Run this version with py server.py. The local server does the internet requests, so browser CORS/API-base problems are avoided. If you installed it with GitHub, the update button runs git pull --ff-only for you.

Set your world, then click Load avg values + sources. The table adds avg value, drop chance %, and two lowest-HP non-boss monster sources in the format Mob(HP). NPCs, bosses, and non-monsters are ignored. If HP values look wrong after an update, click Clear cache once. Loading all 477 rows may take several minutes the first time; cached repeats are faster.

Imbuingi — materiały, Gold Token i porównanie

Gold Token porównanie używa stałych ilości: Basic = 2, Intricate = 4, Powerful = 6. Cena tokena jest edytowalna, więc możesz wpisać aktualną cenę z marketu, nawet jeśli automatyczny lookup jej nie znajdzie.

Notes

  • Market prices are scraped by the local server from the configured public price page source.
  • Drop chances are community statistics from TibiaWiki/Fandom loot sample pages, so treat them as estimates.
  • If an item drops from many monsters, this shows up to 6 sources with the lowest parsed HP first.
  • If a page layout changes, the tool links the relevant source page instead of guessing.
